# Architect Decision

Make a decision on: $ARGUMENTS

## Instructions

As the **Architect** (Opus 4.5), synthesize available information and make a final decision.

Use this command after `/plan`, `/research`, or `/critique` when you need a definitive decision.

### Architect Role
- Synthesize all available inputs
- Resolve conflicts between proposals
- Make clear, justified decisions
- Produce actionable next steps

### Output Format

```
## Decision: {topic}

### Context
{brief summary of inputs considered}

### Decision
{clear statement of what was decided}

### Rationale
{why this approach was chosen}
- Addresses: {key concern 1}
- Addresses: {key concern 2}

### Trade-offs Accepted
- {trade-off 1}

### Next Steps
1. {actionable step}
2. {actionable step}

### Success Criteria
- {how we know this worked}
```

Make the decision now.
