# Critique Task

Critical analysis of: $ARGUMENTS

## Instructions

Spawn the **Critic agent** (openai/gpt-5.1) to critically evaluate the given proposal, plan, or approach.

### Critic Role
- Challenge assumptions
- Identify weaknesses and failure modes
- Validate technical feasibility
- Assess resource requirements realistically
- Propose improvements

### Output Format

```
## Critique: {topic}

### Overall Assessment
{strong|acceptable|needs_work|problematic}

### Blocking Issues
- {critical problem that must be addressed}

### Concerns
- {important but non-blocking issue}

### Suggestions
- {improvement to consider}

### Strengths
- {what works well}

### Verdict
{final recommendation}
```

Execute the critique now.
